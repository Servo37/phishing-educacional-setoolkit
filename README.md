
🎯 Simulação Educacional de Phishing com SEToolkit

📌 Descrição

Este projeto demonstra uma simulação de ataque de phishing utilizando a ferramenta SEToolkit no Kali Linux, com finalidade exclusivamente educacional.

⚠️ Aviso

Este projeto foi realizado em ambiente controlado.
As credenciais utilizadas são fictícias.

🛠️ Ferramentas utilizadas

- Kali Linux
- SEToolkit
- VirtualBox

🚀 Etapas realizadas

1. Execução do SEToolkit
2. Seleção do ataque de engenharia social
3. Clonagem de página (Twitter)
4. Execução do servidor local
5. Acesso via navegador
6. Inserção de credenciais para teste
7. Captura dos dados no terminal

📸 Demonstração

As imagens do processo estão disponíveis na pasta "/images".

🔐 Resultado

- Senha: abcd123

🧠 Aprendizados

- Funcionamento de ataques de phishing
- Importância da engenharia social
- Necessidade de práticas de segurança
